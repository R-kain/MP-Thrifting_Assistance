# MP Thrifting Assistance

중고 상품 사진을 입력받아 외관상 문제점을 분석하고 점수를 산정하기 위한 C++ 프로젝트입니다.

## 현재 상태

이번 초기 구성에서는 다음 내용을 반영했습니다.

- OpenCV 기반 이미지 분석 파이프라인이 들어갈 수 있도록 `ImageInspectionMetrics` 구조체를 정의했습니다.
- 스크래치, 얼룩, 변색, 모서리 손상, 전체 신뢰도 같은 항목으로 점수를 계산하는 `ProductIssueEvaluator`를 추가했습니다.
- 추후 OpenGL 시각화로 확장할 수 있도록 분석 결과를 콘솔 리포트 형식으로 정리했습니다.

## 향후 개발 방향

1. **OpenCV 연동**
   - 사진 로드 및 전처리
   - 윤곽선/오염/스크래치 후보 검출
   - 항목별 정량 지표 추출
2. **점수 모델 고도화**
   - 카테고리별 가중치 분리
   - 학습 데이터 기반 보정
   - 신뢰도 임계값 처리
3. **OpenGL 시각화**
   - 손상 위치 오버레이
   - 항목별 히트맵
   - 최종 품질 점수 대시보드

## 현재 실행 예시

현재 예제 프로그램은 샘플 측정값을 사용해 아래 항목을 출력합니다.

- 총점
- 등급
- 탐지된 문제 목록
- 개선 우선순위 메시지

## 참고

아직 실제 OpenCV/OpenGL 라이브러리 연결은 하지 않았습니다. 현재 구조는 **의존성 연결 전에도 빌드 가능한 초기 골격**이며, 다음 단계에서 Visual Studio 프로젝트에 OpenCV/OpenGL include/lib 설정을 추가하면 됩니다.


## 디렉터리 기반 실행

현재 콘솔 프로그램은 실행 인자로 **이미지 디렉터리 경로**를 받습니다.

```bash
mp_TA.exe C:/images/products
```

- 지원 확장자: `.jpg`, `.jpeg`, `.png`, `.bmp`, `.webp`
- 지정한 폴더 안의 이미지 파일을 순회하며 각 파일마다 리포트를 출력합니다.
- 아직 OpenCV 분석은 연결되지 않아, 현재는 각 이미지에 대해 placeholder metric을 사용합니다.
